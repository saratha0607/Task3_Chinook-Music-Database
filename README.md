# Task3_Chinook-Music-Database
This project analyzes the Chinook music store database using a single SQL query.   The query summarizes sales by genre, demonstrating filtering, aggregation, grouping, joins, and sorting.
# Data Analyst Internship – Task 3

## Objective
To practice basic SQL queries such as filtering, sorting, and aggregation.

## Tools Used
- MySQL
- Superstore Dataset (CSV)

## Files Included
- queries_task3.sql – Contains all SQL queries used in this task
- sales_summary.csv – Exported aggregated results
- superstore.csv – Dataset used for analysis

## SQL Concepts Used
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- HAVING
- BETWEEN
- LIKE
- Aggregate functions (SUM, AVG, COUNT)

## Outcome
Gained confidence in writing basic SQL queries for data analysis.
